# Resource Hub

A curated collection of 175+ Solana development resources across 11 categories, maintained by Superteam Brazil's DevRel team. This is not a generic link dump — every resource here has been evaluated for quality, relevance, and maintenance status. Whether you are writing your first program or optimizing compute units on a production DeFi protocol, this hub provides the tools, education, and references you need.

Superteam Brazil's own open-source projects are integrated throughout where they fit naturally — you will find them in DeFi, Token Standards, Gaming, AI, and Education sections.

## Categories

* [Getting Started](getting-started.md) — The recommended learning path from zero to your first deployed program
* [Courses & Education](courses-and-education.md) — Structured courses, cohort programs, security training, and video content
* [Frameworks & SDKs](frameworks-and-sdks.md) — Program frameworks (Anchor, Pinocchio, Native) and client libraries
* [Development Tools](development-tools.md) — IDEs, CLIs, RPC providers, Actions/Blinks, and payments
* [Testing & Debugging](testing-and-debugging.md) — Unit testing, integration testing, fuzz testing, and block explorers
* [DeFi Development](defi-development.md) — DEX routing, lending, oracles, liquid staking, and vault/stablecoin standards
* [Token Standards](token-standards.md) — SPL Token, Token-2022 extensions, stablecoin specifications, and NFT standards
* [Gaming & Mobile](gaming-and-mobile.md) — Unity SDK, on-chain game engines, mobile wallet adapter, and game AI tooling
* [AI-Assisted Development](ai-assisted-development.md) — solana-claude, MCP servers, AI agent frameworks, and development automation
* [Community & Hackathons](community-and-hackathons.md) — Superteam Brazil, hackathons, grants, and ecosystem communities
* [Open Source References](open-source-references.md) — Production codebases, example programs, and reference implementations
