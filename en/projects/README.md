# Projects

Superteam Brazil builds public goods for the Solana ecosystem. Our open-source work spans developer tooling, protocol standards, education platforms, and reference implementations.

Across 9 GitHub repositories, we maintain 6 primary projects — each targeting a gap in the Solana developer experience.

## Primary Projects

| Project | Description |
|---------|-------------|
| [solana-claude](solana-claude.md) | Claude Code configuration for full-stack Solana development |
| [Superteam Academy](superteam-academy.md) | On-chain learning management system with verifiable credentials |
| [Wiki](wiki.md) | Trilingual knowledge hub for Superteam Brazil |
| [Solana Vault Standard](solana-vault-standard.md) | Standardized vault interface for Solana (sRFC 40) |
| [Solana Stablecoin Standard](solana-stablecoin-standard.md) | Standardized stablecoin issuance and management interface |
| [Solana Game Skill](solana-game-skill.md) | Claude Code skill package for Unity and mobile Solana games |
