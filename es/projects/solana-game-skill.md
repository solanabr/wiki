# Solana Game Skill

**GitHub**: [solanabr/solana-game-skill](https://github.com/solanabr/solana-game-skill)
**Estado**: En desarrollo activo
**Mantenido por**: @kauenet

## Vision General

Un paquete de habilidades de Claude Code para desarrollo de juegos Solana en Unity y dispositivos moviles. Proporciona agentes especializados, comandos y reglas de codigo para construir juegos blockchain — desde el diseno de estado on-chain hasta el renderizado del lado del cliente.

## Caracteristicas

### Reglas Unity/C#

Estandares de codigo .NET aplicados y adaptados para desarrollo de juegos en Unity con integracion Solana:

- Convenciones PascalCase/camelCase para componentes Unity
- Patrones de campos serializados para configuracion del Inspector
- Ordenamiento del ciclo de vida de MonoBehaviour
- Patrones de integracion con Solana.Unity SDK

### Agente Game Architect

Diseno de juegos de alto nivel con conciencia on-chain:

- Planificacion de estado on-chain y diseno de estructura de cuentas
- Integracion de NFTs para activos dentro del juego
- Diseno de economia de tokens
- Arquitectura de game loop con interacciones blockchain

### Agente Unity Engineer

Desarrollo a nivel de implementacion en Unity/C#:

- Patrones de uso del Solana.Unity SDK
- Flujos de conexion de wallet en Unity
- Construccion de transacciones desde clientes C#
- Deserializacion de cuentas para estado del juego

### Agente Mobile Engineer

Desarrollo de juegos moviles multiplataforma:

- React Native con Expo para juegos moviles en Solana
- Integracion con mobile wallet adapter
- Optimizaciones especificas por plataforma para iOS y Android

### Ecosistema PlaySolana

Integracion con la infraestructura de gaming PlaySolana:

- Compatibilidad con PSG1
- Patrones del PlaySolana SDK
- Sincronizacion de estado on-chain multijugador

## Stack Tecnologico

- Claude Code
- Unity
- C#
- React Native
