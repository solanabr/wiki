# Proyectos

Superteam Brazil construye bienes públicos para el ecosistema Solana. Nuestro trabajo open-source abarca herramientas para desarrolladores, estándares de protocolos, plataformas educativas e implementaciones de referencia.

A través de 9 repositorios en GitHub, mantenemos 6 proyectos principales — cada uno apuntando a una brecha en la experiencia del desarrollador de Solana.

## Proyectos Principales

| Proyecto | Descripción |
|---------|-------------|
| [solana-claude](solana-claude.md) | Configuración de Claude Code para desarrollo full-stack en Solana |
| [Superteam Academy](superteam-academy.md) | Sistema de gestión de aprendizaje on-chain con credenciales verificables |
| [Wiki](wiki.md) | Centro de conocimiento trilingüe para Superteam Brazil |
| [Solana Vault Standard](solana-vault-standard.md) | Interfaz de vault estandarizada para Solana (sRFC 40) |
| [Solana Stablecoin Standard](solana-stablecoin-standard.md) | Interfaz estandarizada de emisión y gestión de stablecoins |
| [Solana Game Skill](solana-game-skill.md) | Paquete de skill de Claude Code para juegos en Unity y móvil con Solana |
