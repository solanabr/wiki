## solana-glossary

**GitHub**: [solanabr/solana-glossary](https://github.com/solanabr/solana-glossary)

Um glossário completo de terminologia Solana. Cobre conceitos fundamentais (accounts, programs, PDAs), mecanismos de consenso (Tower BFT, Gulf Stream) e termos específicos do ecossistema (compressed NFTs, token extensions, priority fees).

Útil para onboarding de novos desenvolvedores e como referência para redatores de documentação que precisam de terminologia consistente.

## solana-skills

**GitHub**: [solanabr/solana-skills](https://github.com/solanabr/solana-skills)

Uma coleção de pacotes de skill do Claude Code para desenvolvimento Solana. Cada skill é voltada para um aspecto específico do ecossistema -- desde desenvolvimento de programas até integração de frontend e workflows de deploy.

As skills são modulares e podem ser compostas juntas ou usadas de forma independente, dependendo dos requisitos do projeto.

---

## solana-dev-skill

**GitHub**: [solanabr/solana-dev-skill](https://github.com/solanabr/solana-dev-skill)

A skill base do Claude Code para desenvolvimento Solana em geral. Fornece regras, comandos e configurações de agentes que servem de fundação para o [solana-claude](solana-claude.md).

Esta skill cobre o ciclo de desenvolvimento principal -- build, format, lint, test, deploy -- junto com princípios de segurança e padrões de código específicos para Solana em Rust e TypeScript.
