## What's Inside

### About Superteam Brazil

Superteam Brazil is a collective of builders, developers, and operators accelerating Solana adoption across Latin America's largest tech market. The wiki covers mission, leadership, membership tiers, validator operations, and official links.

### Resources

200+ curated resources across 11 categories: Getting Started, Courses and Education, Frameworks and SDKs, Development Tools, Testing and Debugging, DeFi Development, Token Standards, Gaming and Mobile, AI-Assisted Development, Community and Hackathons, and Open Source References.

### Projects

Six primary open-source projects maintained by Superteam Brazil members: solana-claude, Superteam Academy, Wiki, Solana Vault Standard, Solana Stablecoin Standard, and Solana Game Skill.

### Transparency

Quarterly reports on treasury, spending, deliverables, and community growth.

## Contributing

1. Fork the repository
2. Create a branch: `docs/<description>-DD-MM-YYYY`
3. Edit content in all three language directories (`en/`, `es/`, `pt/`)
4. Open a pull request

All contributions should maintain parity across the three languages.
